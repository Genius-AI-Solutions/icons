# Changelog

## 0.1.0

- initial public release of `@geniusaisolutions/icons`
- framework-agnostic core registry and lookup helpers
- metadata-first icon model with adapter-level custom component escape hatches
- first-class adapters for React, Vue, Solid, Svelte, and Angular
- grouped registries and direct icon exports
- framework-neutral SVG string helpers for vanilla browser and Node usage
- working React, Vue, Solid, Svelte, Angular, vanilla, and Node examples
- release verification scripts, smoke tests, and catalog validation
- open-source repo baseline docs and CI for the first public release
