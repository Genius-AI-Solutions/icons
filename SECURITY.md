# Security Policy

## Supported versions

Security fixes are provided for the latest published release of `@geniusaisolutions/icons`.

## Reporting a vulnerability

Do not open a public GitHub issue for a suspected security vulnerability.

Report it privately to the repository owner:

- https://github.com/Genius-AI-Solutions

Include:

- affected package version
- reproduction steps or proof of concept
- expected impact
- any suggested remediation if you have one

We will review the report, confirm impact, and coordinate a fix and disclosure as appropriate.
