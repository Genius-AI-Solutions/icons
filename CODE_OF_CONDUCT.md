# Code Of Conduct

## Our standard

We expect participants in this project to be respectful, constructive, and professional.
That applies to issues, pull requests, code review, and all other project spaces.

Examples of expected behavior:

- be respectful in technical disagreement
- assume good intent and focus on facts
- give actionable feedback instead of personal criticism
- keep discussions on topic and relevant to the project

Examples of unacceptable behavior:

- harassment, intimidation, or discriminatory language
- personal attacks or insulting remarks
- publishing private information without permission
- spam, trolling, or intentionally disruptive behavior

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, commits, or other contributions that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers through the repository owner on GitHub:

- https://github.com/Genius-AI-Solutions

Security issues should follow the process in [SECURITY.md](./SECURITY.md).
